# Ruby Chatbot
Halo kemarin saya sudah membuat chatbot dengan bahasa python.<br />
Sekarang saya membuatnya dengan bahasa ruby.

# Pemasangan

```php
$ pkg update && pkg upgrade
$ pkg install ruby
$ pkg install git
$ git clone https://github.com/OA-i/ruby-chatbot
$ cd ruby-chatbot
$ ruby chatbot.rb
```
